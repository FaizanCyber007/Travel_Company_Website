# Travel Website (React + Vite + Tailwind)

A modular, accessible travel website following a clean, professional theme.

## Quick Start
```bash
npm install
npm run dev
```

## Tech
- React (Vite)
- Tailwind CSS
- react-router-dom
- react-icons

## Scripts
- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview build

## Color Palette
- Primary: `#81BFDA`
- Light: `#B1F0F7`
- Paper: `#F5F0CD`
- Accent: `#FADA7A`

## Fonts
- Poppins (headings)
- Inter (body)
